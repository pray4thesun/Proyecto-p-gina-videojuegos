PixelZone – Noticias, Eventos y Lanzamientos del Mundo Gamer
PixelZone es una plataforma web dedicada a centralizar noticias, eventos y lanzamientos de videojuegos, diseñada para ofrecer una experiencia moderna, rápida y visualmente atractiva.
El proyecto combina un backend sólido con Node.js  y una interfaz dinámica construida con Pug, todo conectado a una base de datos SQL para gestionar contenido de forma eficiente.

_________________________________________________________________________________________________________

Características principales
Noticias actualizadas sobre videojuegos, estudios, trailers y novedades del sector.

- Eventos gamer: ferias, conferencias, torneos y presentaciones.

- Lanzamientos: próximos juegos, fechas, plataformas y detalles clave.

- Diseño modular con Pug, permitiendo componentes reutilizables (header, footer, cards, etc.).

- Backend en Node.js  + Express, rápido y escalable.

- Base de datos SQL para almacenar juegos, noticias, imágenes, slugs y más.

- Frontend dinámico con JavaScript, animaciones y estilos gamer/neon.

_________________________________________________________________________________________________________

- Tecnologías utilizadas

Backend:	Node.js, Express

Frontend:	HTML, CSS, JavaScript, Pug

Base de datos:	SQL (MySQL / MariaDB)

Estilos:	CSS modular, efectos neon

Otros:	Middleware, rutas dinámicas, controladores
